The IT College Quiz (ITCQ)

A simplistic quiz regarding the IT college

Created as a project for the course Network Applications I.

Current features

  • Basic quiz functionality
  • uiz and admin view as single page applications
  • Admin view allows adding questions and categories
  • MySQL database integration (PHP API)

Current goals

  • Editing of questions in the admin view
  • Session and global statistics
  • Two languages: Estonian and English

Used resources

  • Background pattern base
  • Square icon for 'next' button
  • Notepad icon for TODO image

Helpful sources

  • Google Material Design Guidelines
  • AngularJS Fundamentals In 60-ish Minutes
